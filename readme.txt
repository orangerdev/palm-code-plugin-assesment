This is a plugin for Palm Code Assessment.

== Description ==

The Palm Code Assessment Plugin is designed to handle contact form submissions and store them as custom post types. It includes features such as file uploads, custom post types for services, testimonials, and contact data.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/palm-code-assessment-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings as needed.
